# Neurometal Tool Fuel: Repository Index

This repository has been organized to facilitate automated discovery and ingestion of coding patterns, scripts, and utilities. Below is the categorized structure of the reorganized "Graveyard" scripts.

## Categories

### 1. AI and Machine Learning (`AI_and_Machine_Learning/`)
Tools for synthetic data generation, natural language processing, computer vision, and machine learning models.
- **GenerateSyntheticCustomerDatabase**: Generates realistic synthetic data for testing.
- **Machine Learning**: Collections of house price prediction, clustering, and parameter tuning.
- **natural_language_processing**: Core NLP utilities and scripts.
- **face_detection**: Image-based facial detection tools.

### 2. Web Automation and Scraping (`Web_Automation_and_Scraping/`)
Scripts for automating web interactions and extracting data.
- **AutoLinkedIn**: Automation tools for LinkedIn.
- **Scrapper_Scripts**: Various web scrapers for GitHub and other platforms.
- **Selenium basics with python**: Educational and practical Selenium examples.
- **google_images_scraper**: Tool for bulk downloading images from Google.

### 3. Security and Cryptography (`Security_and_Cryptography/`)
Encryption, decryption, and security utility scripts.
- **Cipher_Scripts**: Implementations of various ciphers (Hill, Caesar, etc.).
- **RSA_Algorithm**: RSA encryption/decryption examples.
- **Password_Generator**: Secure password generation tools.
- **FernetDemo.py**: Symmetric encryption demonstration using cryptography library.

### 4. Data Management and Conversion (`Data_Management_and_Conversion/`)
Utilities for handling and converting between data formats (CSV, JSON, Excel, YAML).
- **CSV-to-Excel / CSV_to_JSON**: Format conversion scripts.
- **Excel_Files_Merger**: Tools to consolidate multiple spreadsheets.
- **json_to_yaml**: Configuration format converters.

### 5. File and System Utilities (`File_and_System_Utilities/`)
Tools for managing local files, directories, and system backups.
- **BulkRename**: Batch file renaming utility.
- **File-Organizer / DirectorySummarizer**: Tools to clean up and analyze local storage.
- **auto_backup**: Automated system or file backup scripts.

### 6. Desktop and GUI Applications (`Desktop_and_GUI_Applications/`)
Interactive games, calculators, and graphical user interface examples.
- **2048 / A-Star-GUI**: Game implementations and algorithm visualizations.
- **Algebra-Solver / GraphicCalculator**: Math-oriented GUI tools.
- **dice_simulator.py**: Simple CLI/GUI simulation.

### 7. Networking and Cloud (`Networking_and_Cloud/`)
Scripts for network discovery, cloud management, and connectivity.
- **EC2-launcher**: AWS EC2 instance management.
- **Port_Scanner / Network_Scripts**: Network diagnostic and discovery tools.
- **azure-openapi**: Integration with Azure cloud services.

### 8. Multimedia Processing (`Multimedia_Processing/`)
Tools for processing Audio, Video, and Image files.
- **Audio/**: Converters, recorders, and text-to-speech tools.
- **Video/**: Downloaders, mergers, and gif creators.
- **Image/**: Compressors, watermarkers, and format converters.

### 9. Programming and Algorithms (`Programming_and_Algorithms/`)
Implementations of data structures, competitive programming solutions, and core algorithms.
- **Data Structures**: Implementations of Trees, Graphs, Lists, etc.
- **algorithms**: Sorting, searching, and optimization algorithm examples.
- **CompetitiveProgramming**: Solutions to problems from HackerEarth, SPOJ, etc.

### 10. Communication and Email (`Communication_and_Email/`)
Automation for messaging platforms and email services.
- **AutoWhatsapp**: Scripts for automating WhatsApp messages.
- **Email_Scripts / SendEmail**: SMTP and IMAP automation.
- **discord_github_bot**: Integration between Discord and GitHub.

---
*Note: This repository contains many other scripts at the root level which serve as a broad "graveyard" of code examples and patterns.*
